Student Management System (Python)

Student Management System is a Python-based desktop application designed to efficiently manage student records using a 'tkinter' GUI and MongoDB as the backend database. It allows users to add, update, delete, and view student details, including fields such as student ID, name, date of birth, gender, email, phone number, address, and a profile picture, stored as binary data in the database. The application features a search function to locate students by ID, a view-all option with, and a theme toggle for light and dark modes. Designed for simplicity and functionality, it is ideal for educational institutions or organizations needing a streamlined way to handle student information.

Live Weather App

Live Weather Appis designed to provide real-time weather updates, forecasts, and air quality information for cities worldwide. It features a user-friendly interface with a dynamic gradient background, an interactive search bar, and a toggle switch for temperature conversion between Celsius and Fahrenheit. Users can fetch weather data by entering a city name or using their current location. The app also includes a 5-day weather forecast, an interactive map with location markers, and a favorites list for quick access to preferred cities. Leveraging the OpenWeatherMap API, it provides weather details such as temperature, humidity, wind speed, pressure, visibility, sunrise and sunset times, and cloudiness.
